## Objective üßê

Write the JavaScript in **js/game.js** so that clicking the cookie:

1. Increases your total cookie count by ‚Äúcookies-per-click.‚Äù  
2. Updates the on-screen **cookieDisplay** and **cpcDisplay** accordingly.  

## Getting Started üö©

### Step 1. Define Your Player State

1. Open **js/game.js**.  
2. At the top, create a simple player object to track cookies and click power:

```js
// js/game.js
const player = {
  cookies: 0,       // total cookies earned
  cookiesPerClick: 1 // cookies gained each click
};
```

### Step 2. Grab DOM Elements

1. Still in game.js, grab references to the display elements and cookie button:

```js
const displayTotal = document.querySelector('.cookieDisplay');
const displayCPC   = document.querySelector('.cpcDisplay');
const cookieBtn    = document.querySelector('.cookieButton');
```

### Step 3. Write the Click Handler

Below those lines, add a click listener to the cookie button:

```js
cookieBtn.addEventListener('click', () => {
  // 1. Increase the player's cookie count
  player.cookies += player.cookiesPerClick;

  // 2. Update the on-screen displays
  displayTotal.textContent = player.cookies;
  displayCPC.textContent   = player.cookiesPerClick;
});
```

## Step 4. Test in the Browser

1. Save game.js and reload index.html in your browser.

2. Click the cookie image:

- cookieDisplay should increment by 1 each click.

- cpcDisplay should show ‚Äú1.‚Äù

3 .Open your developer console (F12) to confirm there are no errors.
