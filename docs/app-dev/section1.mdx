## Objective üßê

Use the supplied CSS rules to style your cookie clicker interface: the save/load buttons, the cookie display, and the shop containers.

## Getting Started üö©

### Step 1. Layout Styles

1. Open **css/style.css** and add:

```css
h1 {
    margin-bottom: 0px;
}
```
- This removes default body margins.

### Step 2. Save/Load Controls

1. Below your base styles, paste:

```css
.saving {
    position: absolute;
    width: 100%;
    left: 5px;
}
.saving button {
    width: 100px;
    float: left;
    border: 2px solid rgb(0, 0, 0);
    border-top: none;
    border-radius: 0px 0px 5px 5px;
    margin-top: -2px;
    height: 27px;
    line-height: 25px;
    outline: none;
}
.saving button.import {
    color: rgb(255, 255, 255);
    background-color: rgb(33, 171, 12);
    border-color: rgb(27, 138, 10);
    box-shadow: 0px 3px rgb(18, 87, 7);
    border-right: none;
    border-radius: 0px 0px 0px 5px;
}
.saving button.export {
    color: rgb(255, 255, 255);
    background-color: rgb(158, 29, 29);
    border-color: rgb(136, 25, 25);
    box-shadow: 0px 3px rgb(92, 16, 16);
    border-left: none;
    border-radius: 0px 0px 5px 0px;
}
.saving button.export.wipe {
    position: absolute;
    right: 10px;
    border-radius: 0px 0px 5px 5px;
    border-left: 2px solid rgb(136, 25, 25);
}
```
- These rules style the top‚Äêbar buttons, including hover/active shadows (hover/active rules come later).

### Step 4. Cookie & Counter

1. Next, style the main game area:

```css
.cookieContainer {
    width: 100%;
    height: 310px;
    text-align: center;
}
.cookieButton {
    background-color: transparent;
    border: none;
    outline: none;
    padding: 0;
    margin: auto;
    display: block;
}
.cookieButton img {
    width: 300px;
    height: 300px;
    transition: transform 0.1s ease;
}
.cookieButton img:hover {
    transform: translateY(2.5px);
}
.cookieButton img:active {
    transform: translateY(5px);
}
.cookieDisplay {
    font-size: 24px;
    user-select: none;
}
.cpsDisplay, .cpcDisplay {
    font-weight: bold;
    user-select: none;
}
```
- This centers your cookie, adds click animations, and styles the count displays.
