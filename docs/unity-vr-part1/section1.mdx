# Part 1: Install Unity & Create a New Project

## 1. Install Unity & Create a New Project

Download Unity Hub (if you haven’t already) from [unity.com/download](https://unity.com/download).

Use Unity Hub to install **Unity 2020.3 LTS** (or any version 2020.3 or later).

In Unity Hub, click **New project**:
- **Select** the **3D (Universal Render Pipeline)** template (makes lighting nicer).
- **Name** your project **“VR Basketball Maze”**.
- **Choose** a folder on your computer (e.g., `C:/Users/You/Documents/UnityProjects/VR_Basketball_Maze`).
- **Click** **Create**.

Unity will open a blank 3D project. It may take a minute to load.
